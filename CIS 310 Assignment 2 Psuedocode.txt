CIS 310 Assignment 2 Pseudocode Overview:
SETUP string name
SETUP data for size of string
PRINT name

FIRST LOOP:
	PUSH name character by character onto eax stack

SECOND LOOP:
	POP eax and MOVE name to al register, reversing string

PRINT reversed string
